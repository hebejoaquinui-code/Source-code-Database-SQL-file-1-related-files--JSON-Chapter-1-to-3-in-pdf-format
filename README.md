The purpose of our project is to help Donadel Hometel manage their reservations more easily and efficiently.
Right now, they use manual booking and different platforms like Facebook and Airbnb, which sometimes causes missed bookings, 
delayed replies, and overbooking. The system will provide a centralized platform for booking and monitoring reservations.
It aims to reduce missed bookings, delayed responses, and overbooking issues. 
The system will also generate reports and occupancy analytics to help management understand booking trends.
Overall, the project is designed to improve reservation management and support better decision-making.
